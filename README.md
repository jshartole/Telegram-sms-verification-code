Telegram 平台可以通过不同的方式将此代码发送给用户：

有时，用户确实知道没有其他会话（例如，他只是手动终止了它们），但应用程序仍然报告代码已发送到另一个会话。在这种情况下，等待几十分钟或几个小时通常会有所帮助。
